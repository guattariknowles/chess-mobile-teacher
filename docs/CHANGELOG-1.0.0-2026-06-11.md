# 自由象棋 1.0.0 测试版

构建日期：2026-06-11

## 已实现

- 离线双人国际象棋棋盘。
- 点击选择棋子和点击目标格走棋。
- 合法走法高亮和非法走法拦截。
- 将军、将死与和棋状态提示。
- 王车易位、吃过路兵和兵升变。
- 悔棋、重新开始和棋盘翻转。

## 尚未实现

- PGN 保存、导入和回放。
- 教学、AI 和 Stockfish 分析。
