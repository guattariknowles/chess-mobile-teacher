# 自由象棋 1.1.0 测试版

构建日期：2026-06-11

## 新增功能

- 黑方和白方独立棋钟与操作区。
- 无棋钟、固定时间和自定义时间模式。
- 每步加秒、暂停和继续。
- 双方独立认输与悔棋按钮。
- 超时自动判负。
- 将死、和棋、认输和超时后的结果弹窗。
- 回顾棋局占位页面，明确说明 PGN 回放和分析尚未接入。

## 安装说明

- 旧版 `1.0.0` 使用 EAS 云端签名。
- 新版 `1.1.0` 使用本机测试签名。
- 两个版本签名不同，Android 不允许直接覆盖安装。
- 安装 `1.1.0` 前需要先卸载旧版。
- 当前版本尚未保存棋谱或用户资料，因此卸载旧版不会丢失这类数据。
